# Online Exam System (C++ OOP Project)

## Overview
This project is an Online Exam System developed using C++ and Object-Oriented Programming concepts.

## Features
- Teacher Interface
- Student Interface
- MCQ Exam System
- Add Exams
- Attempt Exams
- View Scores
- Dynamic Memory Allocation
- Virtual Functions
- Inheritance

- OOP
